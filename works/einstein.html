Eintein Maps

Descrição Oficial

Official Description

Aplicativo com sistema integrado de navegação interna para ser usado dentro do hospital Albert Einstein. Com ele o usuário poderá acessar todos os blocos e andares do prédio e ficar por dentro das últimas notícias.

App with an internal navigation system to be used within Albert Einstein hospital. The user has access to every block and every floor on the buildings and can follow the latest news.

Desenvolvimento

Development

O aplicativo foi desenvolvido com Unity3D baseado em uma versão anterior em C++. Minha responsabilidade foi montar todas as interfaces e sistemas, exceto pelo sistema de navegação e pathfinding.

The app was developed using Unity3D based on an older C++ version. My job was to put every interface and system together, except for the navigation and pathfinding systems.

Além de usar a Unity3D como engine, a linguagem que escolhemos foi C# e usamos NGUI para montar as interfaces. As linhas no mapas foram feitas usando Vectrosity e os mapas em sí também usam um plugin para o pathfinding. Refazer o aplicativo demorou em torno de duas semanas.

Besides Unity3D being the engine, the language we used was C# and NGUI for the interface. The lines on the maps were done using Vectrosity and the maps also use a plugin for pathfinding. It took around two weeks to finish the new version of the app.
